# Sarif-viewer

## Описание

Программа позволяет анализировать сработки из SAST в sarif файле в удобном UI.

## Возможности

1. Открывание разных SARIF файлов
2. Сохранение прогресса работы в JSON файл (генерируется рядом с sarif_viewer_vN.py)
3. Загрузка разных JSON состояний и их сравнение на отличия
4. Присваивание каждой сработке своей критичности (`Critical`, `High`, `Middle`, `Low`, `Error`, `Info`) и статуса (`Отклонено`, `Подтверждено`, `Не определено`)
5. Генерация отчета по найденным уязвимостям в стиле AppSec
6. Темная тема (experimental)

## Примечания

После генерации отчета таблица будет выглядеть не так, как должна. Для этого надо ее полностью выделить, нажать по ней ПКМ, выбрать `Выровнять ширину столбцов и после этого расширить таблицу` и поменять ширину каждого столбца.
